Introducción

La base de datos de PrestaShop se organiza en múltiples tablas, cada una con un rol específico en la administración de la tienda en línea. A continuación, se presenta una descripción detallada de las tablas principales y cómo estas se integran dentro del sistema para administrar productos, clientes y pedidos.

1. Tablas de Productos

1.1 ps_product

La tabla ps_product almacena información general de cada producto, como:

  • ID: Identificador único del producto.
  • Precio: Precio base del producto.
  • Referencia: Código de referencia o SKU.
  • Estado: Indica si el producto está activado o desactivado.
  • Visibilidad: Define si el producto es visible en el catálogo.

1.2 ps_product_lang

Esta tabla contiene información específica para cada idioma sobre el producto, como:

  • Nombre del Producto: Nombre del producto en cada idioma.
  • Descripción: Detalles descriptivos del producto.
  • Etiquetas: Palabras clave o tags asociados al producto.

1.3 ps_product_attribute

Gestiona las combinaciones de productos (por ejemplo, color y tamaño), incluyendo el precio y las existencias para cada variación.

2. Tablas de Clientes

2.1 ps_customer

Almacena los datos personales de cada cliente, incluyendo:

  • Email: Correo electrónico del cliente, usado como usuario de acceso.
  • Contraseña: Contraseña cifrada para el inicio de sesión seguro.
  • Fecha de Registro: Fecha en que el cliente se registró.

2.2 ps_customer_group

Define los grupos a los que pertenece cada cliente, permitiendo segmentación para precios o descuentos personalizados.

3. Tablas de Pedidos

3.1 ps_orders

Esta tabla rastrea cada pedido realizado en la tienda, incluyendo:

  • ID de Cliente: Identifica al cliente que realizó el pedido.
  • Fecha de Creación: Fecha en que se realizó el pedido.
  • Estado del Pedido: Indica el estado actual, como pendiente, procesado o enviado.

3.2 ps_order_detail

Almacena los artículos específicos de cada pedido, incluyendo cantidad, precio por unidad y detalles de productos asociados.

3.3 ps_cart

Esta tabla registra los carritos de compra de los clientes, permitiendo almacenar productos seleccionados antes de realizar la compra final.

4. Otras Tablas Relevantes

4.1 ps_category

Define las categorías de productos en la tienda, facilitando la organización y navegación para los usuarios.

4.2 ps_stock_available

Gestiona el inventario disponible para cada producto o combinación de producto, incluyendo alertas de bajo stock.

4.3 ps_currency

Contiene las monedas configuradas en la tienda y sus respectivas tasas de cambio, permitiendo a los clientes realizar transacciones en su moneda preferida.
